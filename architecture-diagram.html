Data Flow per Scenario
Step 1
Scenario Starts
  • PicoContainer creates TestContext
  • @Before hooks fire (order 0 ‚Üí 1)
  • Scenario name registered
  • Playwright tracing starts
Step 2
Browser Opens (Lazy)
  • First getPage() call opens browser
  • DriverFactory launches Playwright
  • BrowserContext created with video
  • ElementUtils wired to Page
Step 3
Steps Execute
  • Step calls Page Object method
  • Page Object uses ElementUtils
  • ElementUtils tries selectors in order
  • Test data fetched from Excel cache
Step 4
Scenario Ends
  • @After hooks fire (order 3 ‚Üí 0)
  • Screenshot / trace captured if failed
  • Video saved or deleted
  • Browser closed, ThreadLocal cleared
